✅ ESTADO ACTUAL DEL PORTAL

Las variables se activan automáticamente por fecha — no necesitas hacer nada para eso.
Para que el Chat IA y las descargas funcionen, sigue las instrucciones de los pasos abajo.

1
Publicar en GitHub Pages (gratis, 5 minutos)
1. Ve a github.com y crea una cuenta gratuita (si no tienes).
2. Crea un repositorio nuevo llamado servimine-portal (público).
3. Sube este archivo HTML como index.html.
4. Ve a Settings → Pages → Source: main branch → Save.
5. Tu portal quedará en: https://TU-USUARIO.github.io/servimine-portal

Cada vez que edites el archivo y lo subas a GitHub, los alumnos ven los cambios al instante.
2
Activar el Chat IA — Clave API via Workers (gratis)
El Chat necesita una clave API de Anthropic. No se puede poner directamente en el HTML (cualquiera la vería).
La solución es un proxy gratuito en Cloudflare Workers:

1. Ve a workers.cloudflare.com y crea cuenta gratuita.
2. Crea un Worker nuevo y pega este código:
export default { async fetch(request, env) { if (request.method === 'OPTIONS') { return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'POST', 'Access-Control-Allow-Headers': 'Content-Type', } }); } const body = await request.json(); const resp = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'x-api-key': env.ANTHROPIC_KEY, 'anthropic-version': '2023-06-01' }, body: JSON.stringify(body) }); const data = await resp.json(); return new Response(JSON.stringify(data), { headers: { 'Content-Type': 'application/json', 'Access-Control-Allow-Origin': '*' } }); } };

   Nombre: ANTHROPIC_KEY  ·  Valor: tu clave API de Anthropic
4. Copia la URL de tu Worker (ej: https://mi-proxy.mi-usuario.workers.dev)
5. Pégala en el campo de abajo y guarda el portal.
